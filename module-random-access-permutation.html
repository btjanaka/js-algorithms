(inner) RAPNonUnique(arr, n) → {array}

Calculates the n-th lexicographical permutation of the items in the array. Does not take into account repeated permutations. Complexity: O(k|arr|2), where k represents the time to carry out bigint operations.
Parameters:
Name Type Description
arr array A sorted array to find the permutation of
n JSBI.BigInt The number of the permutation to find (0-based)
